# 🛒 Amazon Clone | E-Commerce Frontend UI

A modern **Amazon-inspired e-commerce frontend clone** built using **HTML5** and **CSS3** to practice responsive layouts, Flexbox, CSS Grid, and real-world UI recreation.

This project focuses entirely on frontend development and recreates the shopping interface for educational purposes.

> **Educational Portfolio Project**
>
> This project is a frontend practice exercise created solely for learning and portfolio purposes. It is **not affiliated with, endorsed by, or connected to Amazon**. No user authentication, payment processing, or backend services are implemented.
